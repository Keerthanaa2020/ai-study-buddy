# AI Study Buddy

An AI-powered web application that simplifies study topics and generates quizzes to test student understanding.

## Features
- Topic simplification using AI
- Automatic quiz generation
- Instant scoring and feedback
- User-friendly interface

## Technologies Used
- React.js
- Tailwind CSS
- JavaScript
- AI Language Model

## How to Run
1. Install dependencies:
   npm install
2. Start the app:
   npm start
